Пропуска МосПолитеха

Сервис для заказа пропусков на территорию Московского Политеха

Проверить пропуск

Этапы заказа пропуска

1. Позвоните в дирекцию. Человек, которому требуется пропуск на проход в определенную локацию Московского Политеха должен позвонить в нужную для него дирекцию и заказать пропуск

1/3

2. Заказ пропуска через сервис. Сотрудник Московского Политеха, согласно внутренним правилам, при возможности допустить человека на локацию, заказывает пропуск, через сервис.

2/3

3. Получить пропуск. Сотрудник бюро пропусков, после необходимых проверок, согласно внутренним правилам учебного заведения, выпускает пропск и выдает его человеку

3/3
